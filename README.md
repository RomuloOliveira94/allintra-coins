<h2 id="started">🚀 Instalação</h2>

A instalação é simples

<h3>Prerequisites</h3>

- [NodeJS](https://nodejs.org/en)
- [Git](https://git-scm.com/)

<h3>Clonar</h3>

```bash
git clone https://github.com/RomuloOliveira94/allintra-coins.git
```

<h3>Iniciando o projeto</h3>

```bash
cd allintra-coins
npm install
npm run dev
```
